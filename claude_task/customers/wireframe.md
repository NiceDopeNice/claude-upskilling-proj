## Search Bar — Simple Mode (default)

```
┌─────────────────────────────────────────────────────────────────────┐
│  [Name ▾]  [Search by name................................] [⊞]    │
└─────────────────────────────────────────────────────────────────────┘

  [Name ▾] = multi-select column dropdown (checkboxes, stays open)
  [⊞]     = switch to advanced search
```

**Column multi-select dropdown (stays open on check/uncheck):**
```
┌─────────────┐
│ ✓ Name      │  ← checked
│   Customer #│
│   Email     │
│   Phone     │
│   SSN       │
└─────────────┘
```

**Two columns selected:**
```
┌──────────────────────────────────────────────────────────────────────┐
│  [2 columns ▾]  [Search by name, email................] [⊞]        │
└──────────────────────────────────────────────────────────────────────┘

  "john" in Name, Email  ×
```

**No column selected warning:**
```
┌──────────────────────────────────────────────────────────────────────┐
│  [Select columns ▾]  [john............................] [⊞]        │
└──────────────────────────────────────────────────────────────────────┘
  ⚠ Please select at least one column
```

## Search Bar — Multi Mode (advanced)

```
┌─────────────────────────────────────────────────────────────────────┐
│  ⠿ Customer No [__________] ×   ⠿ SSN [__________] ×             │
│  ⠿ Name        [__________] ×   ⠿ Phone [_________] ×            │
│  ⠿ Email       [__________] ×   [+ Add filter ▾]         [⊟]    │
└─────────────────────────────────────────────────────────────────────┘

  ⠿ = drag handle (reorderable)
  ×  = remove chip
  [⊟] = switch to simple search
```

**Add filter dropdown:**
```
┌──────────────────┐
│   Address        │
│   Alt. Email     │
└──────────────────┘
```
